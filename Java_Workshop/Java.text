Cursor

A cursor is an object that is used to traverse (iterate through) the elements of a collection.

Types of Cursors
                  1. Iterator:
                              Used to traverse any collection that implements the Collection interface.
                              Can move only in the forward direction.
                              Can read and remove elements during traversal.

                  2. ListIterator:
                                    Used only with classes that implement the List interface.
                                    Can move in both forward and backward directions.
                                    Can read, add, update, and remove elements during traversal.

                  3. Enumeration:
                                    Used with legacy classes such as Vector and Hashtable.
                                    Can move only in the forward direction.
                                    Can only read elements; it cannot add, update, or remove them.


HashMap
      Child of the Map interface.
      Introduced in Java 1.2.
      Stores data in key-value pairs.
      Keys are unique, but values can be duplicate.
      Allows one null key and multiple null values.
      Order of elements is not maintained.
      Not synchronized (not thread-safe).
